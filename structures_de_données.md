## Liste python :
`>>> li = [7,1,9,"a"]`
* lire :
  ```python
  >>> li[2]
   9
* écrire :
  ```python
  li[3] = 12
  >>> print(li)
  [7, 1, 12, "a"]

* append ajoute un élément à la fin de la liste. :
  ```python
  >>> li = [7, 1, 9, "a"]
  >>> li.append("banane")   # ajoute str "banane" à la fin
  >>> print(li)
  [7, 1, 9, "a", "banane"]
